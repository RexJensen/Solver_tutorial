# CFR Poker Solver: From First Principles to NLHE

A step-by-step tutorial building a Counterfactual Regret Minimization solver,
from Rock-Paper-Scissors to No-Limit Hold'em.

## Curriculum

| Lesson | File | Game | Concepts |
|--------|------|------|----------|
| 1 | `lesson_01_rps.py` | Rock-Paper-Scissors | Nash Equilibrium, Regret Matching, Exploitability |
| 2 | *coming soon* | Kuhn Poker | CFR, Game Trees, Information Sets |
| 3 | *coming soon* | Leduc Poker | Multi-street CFR, Betting Rounds |
| 4 | *coming soon* | NLHE Abstractions | Card/Action Abstraction, Bucketing |
| 5 | *coming soon* | NLHE Solver | CFR+ / MCCFR for Real Poker |

## Running

```bash
pip install numpy
python lesson_01_rps.py
```
